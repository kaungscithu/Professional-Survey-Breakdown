# Professional-Survey-Breakdown
This dashboard provides a comprehensive breakdown of a professional survey focused on the data industry, covering demographics, compensation, and career sentiment.

Data Professional Survey Dashboard
Project Overview
This project features an interactive dashboard that analyzes the career landscape for data professionals globally. Using a dataset of 630 survey respondents, the dashboard explores the intersection of job titles, programming preferences, compensation, and workplace satisfaction.

Key Insights
The Python Dominance: Python is the most preferred language across all data roles, significantly outperforming R and JavaScript.

The Compensation Gap: Despite competitive salaries for Data Scientists and Engineers, the Happiness-Salary score (4.27/10) is notably lower than the Work/Life Balance score (5.74/10), indicating that many professionals feel undercompensated.

Market Entry: Over 67% of respondents find entering the data field to be either "Difficult" or "Neither easy nor difficult," highlighting the competitive nature of the industry for newcomers.

Tech Stack & Skills Demonstrated
Data Visualization: Power BI / Tableau (or the specific tool used to create the charts).

Data Analysis: Interpreting complex demographic and financial trends.

Tooling: Categorizing data by geography (US, India, UK, etc.) and professional hierarchy.

Actionable Reporting: Turning raw survey responses into high-level executive summaries.

Dashboard Components
Demographics: Tracks the average age (29.87) and geographic distribution of the workforce.

Salary Analysis: Breaks down average pay by job title and provides a comparative look at salary by sex.

Sentiment Gauges: Visualizes Work/Life balance and Salary Satisfaction using radial gauges.

Language Popularity: A stacked bar chart showing which programming languages are favored by specific job titles.
